# Download Torrent

Submit a magnet link and voila!

**Not for piracy!**
Please use it for legitimate distributed file sharing.

To start:
```
npm install
node index.js
```

Go to your browser. Default port 3001
```
http://localhost:3001
```

Customization via Environment Variables
```
PORT=8080
DOWNLOAD_PATH=<ABSOLUTE_PATH>
```
